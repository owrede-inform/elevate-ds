# ELEVATE Design System

INFORM's design system for building consistent, accessible interfaces across all products.

## What You Get

- **45+ Components** - Buttons, forms, tables, navigation, data display
- **Design Tokens** - Colors, typography, spacing, themes  
- **Patterns** - Common layouts and application shells
- **Accessibility** - WCAG 2.1 AA compliance built-in
- **Theming** - Light, dark, and custom brand support
