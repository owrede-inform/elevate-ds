## Usage Tips

### Best Practices
- **Consistent Spacing**: Use CSS or layout components for consistent gaps between elements
- **Proper Labeling**: Always provide `label` attributes for form inputs for accessibility
- **Button Hierarchy**: Use `tone="primary"` for main actions, `tone="neutral"` for secondary actions
- **Card Padding**: Use `padding` attribute values: `s`, `m`, `l`, `xl` for consistent spacing
- **Web Standards**: ELEVATE components are native web components that work in any framework or vanilla HTML

### Accessibility Features
- **Keyboard Navigation**: All components support keyboard interaction
- **Screen Reader Support**: Proper ARIA attributes and semantic markup
- **Focus Management**: Clear focus indicators for all interactive elements
