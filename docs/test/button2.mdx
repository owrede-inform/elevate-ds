## API Reference

Based on `elvt-button` component API:

### Properties
- `tone`: `neutral` (default), `primary`, `success`, `warning`, `danger`, `emphasized`, `subtle`
- `size`: `small`, `medium` (default), `large`
- `disabled`: Boolean to disable interaction
- `selected`: Display as selected state
- `href`: Renders as anchor tag when provided
- `target`: Link target when href is present
- `type`: `button` (default), `reset`, `submit`

### Slots
- `default`: Button's label/content
- `prefix`: Icon or element before text
- `suffix`: Icon or element after text
