# Lightbox

Lightbox provides an overlay interface for viewing media content like images and videos in a focused, modal environment with zoom controls, navigation, and keyboard support.

## When to Use

- **Image Galleries**: View images in full size with navigation
- **Media Preview**: Preview videos, documents, or other media
- **Product Images**: Detailed product image viewing in e-commerce
- **Portfolio Display**: Showcase design work or photography
