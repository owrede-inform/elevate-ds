# Tooltip

Tooltip component provides contextual information that appears on hover or focus, offering additional details, help text, or explanations without cluttering the interface.

## When to Use

- **Help Text**: Additional context for form fields or UI elements
- **Definitions**: Explanations of terms or concepts
- **Status Information**: Details about system states or processes
- **Action Descriptions**: Clarification for icon buttons or controls

:::tip Accessibility
Tooltips automatically include proper ARIA attributes for screen readers. Keep tooltip content concise and helpful.
:::

:::info Timing
Tooltips appear after a short delay on hover and disappear when the cursor moves away or focus is lost.
:::
