# Icon Button

Icon Button is a compact button variant that displays only an icon without text, making it perfect for toolbars, action bars, and interfaces where space is limited but actions need to be easily accessible.

<TocLinks />

## When to Use

- **Toolbars**: Action buttons in editing interfaces or toolbars
- **Space-Constrained**: When space is limited but actions are needed
- **Secondary Actions**: Quick actions that don't need prominent text labels
- **Navigation**: Back, close, menu, and navigation actions

## Framework Integration

### React Usage

```jsx
import { ElvtIconButton } from '@inform-elevate/elevate-core-ui/dist/react';

function Toolbar() {
  return (
    <div className="toolbar">
      <ElvtIconButton icon="menu" ariaLabel="Open menu" onClick={openMenu} />
      <ElvtIconButton icon="search" ariaLabel="Search" onClick={openSearch} />
      <ElvtIconButton icon="settings" ariaLabel="Settings" onClick={openSettings} />
    </div>
  );
}
```

---

:::tip Accessibility
Always provide `aria-label` or `aria-labelledby` attributes for icon buttons since they have no visible text. Screen reader users need to understand what each button does.
:::
