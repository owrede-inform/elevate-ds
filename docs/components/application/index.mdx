# Application

The Application component serves as the root wrapper for ELEVATE applications, providing essential setup including theme context, global styles, CSS custom properties, and foundational configuration.

<TocLinks />

## When to Use

- **Root Component**: Wrap your entire application at the top level
- **Theme Setup**: Enable ELEVATE theming and design tokens
- **Global Styles**: Apply ELEVATE global styles and CSS reset
- **Context Providers**: Initialize ELEVATE context providers

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `theme` | `'light' \| 'dark' \| 'auto'` | `'auto'` | Sets the application theme mode |
| `compact` | `boolean` | `false` | Enables compact spacing mode |
| `dir` | `'ltr' \| 'rtl'` | `'ltr'` | Sets text direction for internationalization |

## Slots

| Slot | Description |
|------|-------------|
| Default | Main application content |

## Framework Integration

### React Usage

```jsx
import { ElvtApplication } from '@inform-elevate/elevate-core-ui/dist/react';

function App() {
  return (
    <ElvtApplication theme="light">
      <div>
        <h1>My App</h1>
        {/* Your app content */}
      </div>
    </ElvtApplication>
  );
}
```

## Features

### Theme Management
- **Automatic Theme Detection**: Respects system theme preferences
- **Manual Override**: Set specific light or dark themes
- **CSS Custom Properties**: Provides all ELEVATE design tokens
- **Theme Switching**: Runtime theme changes with smooth transitions

### Global Styles
- **CSS Reset**: Normalized base styles across browsers
- **Typography**: Default font families and text styles
- **Focus Management**: Consistent focus indicators
- **Responsive**: Mobile-first responsive utilities

### Accessibility
- **Screen Reader Support**: Proper semantic structure
- **High Contrast**: Supports high contrast mode
- **Reduced Motion**: Respects motion preferences
- **Keyboard Navigation**: Focus management and tab order

## Best Practices

### Setup
```jsx
// ✅ Good - Wrap entire application
function App() {
  return (
    <ElvtApplication>
      <Router>
        <Routes>
          <Route path="/" element={<Home />} />
          <Route path="/about" element={<About />} />
        </Routes>
      </Router>
    </ElvtApplication>
  );
}

// ❌ Avoid - Multiple Application components
function App() {
  return (
    <div>
      <ElvtApplication>
        <Header />
      </ElvtApplication>
      <ElvtApplication>
        <Main />
      </ElvtApplication>
    </div>
  );
}
```

### Theme Configuration
```jsx
// ✅ Good - Let system preferences control theme
<ElvtApplication theme="auto">

// ✅ Good - Specific theme for branded apps
<ElvtApplication theme="light">

// ✅ Good - RTL support
<ElvtApplication dir="rtl" theme="auto">
```

:::tip Theme Integration
The Application component automatically applies ELEVATE design tokens as CSS custom properties, making them available to all child components and custom styles.
:::

:::info Global Setup
Only use one Application component per app, typically at the root level. It provides global context and styles that should wrap your entire application.
:::
