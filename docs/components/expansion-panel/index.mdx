# Expansion Panel

Expansion Panels provide collapsible content sections that help organize and manage information density. They allow users to show and hide detailed content on demand, making interfaces more scannable and less overwhelming.

## When to Use

- **Content Organization**: Group related information into expandable sections
- **Progressive Disclosure**: Show summary first, details on demand
- **Space Conservation**: Reduce visual clutter in dense interfaces
- **FAQ Sections**: Organize frequently asked questions and answers

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `expanded` | `boolean` | `false` | Whether the panel is expanded by default |
| `disabled` | `boolean` | `false` | Disables interaction with the panel |
| `icon` | `string` | | Icon name to display in the header |
| `variant` | `'default' \| 'bordered' \| 'elevated'` | `'default'` | Visual style variant |
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | Overall size of the panel |

## Slots

| Slot | Description |
|------|-------------|
| `header` | Main header content (required) |
| `description` | Optional description text below the header |
| `content` | Collapsible content area (required) |
| `icon` | Custom icon content (overrides icon property) |

## Events

| Event | Detail Type | Description |
|-------|-------------|-------------|
| `elvt-expand` | `CustomEvent<{ expanded: boolean }>` | Fired when panel expands or collapses |
| `elvt-before-expand` | `CustomEvent<{ expanded: boolean }>` | Fired before expansion state changes (cancelable) |

## Framework Integration

### React Usage

```jsx
import { ElvtExpansionPanel } from '@inform-elevate/elevate-core-ui/dist/react';

function SettingsPage() {
  const [expandedPanels, setExpandedPanels] = useState(['notifications']);

  const handleExpand = (panelId, event) => {
    if (event.detail.expanded) {
      setExpandedPanels([...expandedPanels, panelId]);
    } else {
      setExpandedPanels(expandedPanels.filter(id => id !== panelId));
    }
  };

  return (
    <div>
      <ElvtExpansionPanel
        expanded={expandedPanels.includes('general')}
        onElvtExpand={(e) => handleExpand('general', e)}
      >
        <span slot="header">General Settings</span>
        <div slot="content">
          <GeneralSettings />
        </div>
      </ElvtExpansionPanel>
      
      <ElvtExpansionPanel
        icon="bell"
        expanded={expandedPanels.includes('notifications')}
        onElvtExpand={(e) => handleExpand('notifications', e)}
      >
        <span slot="header">Notifications</span>
        <span slot="description">Configure your notification preferences</span>
        <div slot="content">
          <NotificationSettings />
        </div>
      </ElvtExpansionPanel>
    </div>
  );
}
```

### Vue Usage

```vue
<template>
  <elvt-expansion-panel
    :expanded="isExpanded"
    @elvt-expand="handleExpand"
    icon="settings"
  >
    <span slot="header">Advanced Settings</span>
    <span slot="description">Configure advanced features and options</span>
    <div slot="content">
      <AdvancedSettings />
    </div>
  </elvt-expansion-panel>
</template>

<script>
export default {
  data() {
    return {
      isExpanded: false
    };
  },
  methods: {
    handleExpand(event) {
      this.isExpanded = event.detail.expanded;
    }
  }
};
</script>
```

## Accessibility

- **Keyboard Navigation**: Spacebar and Enter keys toggle expansion
- **ARIA Attributes**: Proper `aria-expanded` and `aria-controls` attributes
- **Screen Reader Support**: Announces expansion state changes
- **Focus Management**: Focus moves logically through panel content

## Best Practices

### Content Organization

```jsx
// ‚úÖ Good - Logical grouping
<ElvtExpansionPanel>
  <span slot="header">Account Security</span>
  <span slot="description">Password, 2FA, and login settings</span>
  <div slot="content">
    <SecuritySettings />
  </div>
</ElvtExpansionPanel>

// ‚ùå Avoid - Unrelated content
<ElvtExpansionPanel>
  <span slot="header">Mixed Settings</span>
  <div slot="content">
    <PasswordField />
    <ColorPicker />
    <NotificationToggle />
  </div>
</ElvtExpansionPanel>
```

### Header Content

```jsx
// ‚úÖ Good - Clear and descriptive
<ElvtExpansionPanel icon="credit-card">
  <span slot="header">Payment Methods</span>
  <span slot="description">Manage cards, billing, and payment preferences</span>
</ElvtExpansionPanel>

// ‚ùå Avoid - Vague headers
<ElvtExpansionPanel>
  <span slot="header">Other</span>
  <span slot="description">Stuff</span>
</ElvtExpansionPanel>
```

### Performance

```jsx
// ‚úÖ Good - Lazy load heavy content
<ElvtExpansionPanel>
  <span slot="header">Analytics Dashboard</span>
  <div slot="content">
    {isExpanded && <AnalyticsDashboard />}
  </div>
</ElvtExpansionPanel>
```
