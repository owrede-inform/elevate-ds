# Documentation Components

Custom React components built for this documentation site to visualize and demonstrate ELEVATE design system elements.

:::info Site-Specific Components
These components are **not** part of the ELEVATE Core UI library. They are documentation tools only.
:::

<TocLinks />

## Available Components

### [ColorRamp](./components/colorramp/)
Displays ELEVATE color tokens with pattern matching and multiple templates.

### [ComponentShowcase](./components/componentshowcase/)
Interactive component previews with multi-framework code generation.

## Usage

```tsx
import ComponentName from '@site/src/components/ComponentName';
```

All components include API references, examples, and usage guidelines in their individual documentation pages.
